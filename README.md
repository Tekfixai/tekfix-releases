# LanmarkTekFix Releases

Public release binaries for the LanmarkTekFix Support Agent.

## Install via winget

```powershell
winget install Lanmark.TekFixSupportAgent
```

## Silent Install (RMM Deployment)

```powershell
LanmarkTekFixSetup_v3.1.2_Silent.exe /VERYSILENT /SUPPRESSMSGBOXES /NORESTART
```

## Silent Uninstall

```powershell
"C:\Program Files\Lanmark\LanmarkTekFix Support Agent\unins000.exe" /VERYSILENT /SUPPRESSMSGBOXES /NORESTART
```

## About

LanmarkTekFix Support Agent provides AI-powered IT troubleshooting for Windows devices, with seamless escalation to live Lanmark engineers.

Published by [Lanmark Limited](https://www.lanmark.co.uk)
